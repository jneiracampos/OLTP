Pasos a seguir para la instalación de la aplicación: 

1. Descromprima la carpeta .zip
2. Importe el proyecto completo en un ambiente de desarrollo de su preferencia
3. Ejecutar las pruebas de conexión ConexionTest.java
4. Conexión a la base de datos en el archivo src/main/resources/META-INF/persistence.xml
5. Cree las tablas de la base de datos ejecutando el script ubicado en data/EsquemaSuperAndes.sql
6. Ejecutar nuevamente las pruebas de conexión
7. Explorar InterfazSuperAndesApp.java
